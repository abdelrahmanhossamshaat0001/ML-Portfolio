# 🩺 Diabetes Prediction

## 📌 Overview
This project applies machine learning techniques to predict whether a patient is likely to have diabetes based on medical attributes.

## 📂 Dataset
- Dataset: Diabetes Dataset
- Target: Diabetes Outcome

## 🛠 Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Project Tasks
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Feature Engineering
- Model Training
- Model Evaluation

## 🚀 How to Run
1. Install the required Python libraries.
2. Open the notebook in Jupyter Notebook.
3. Run all notebook cells.
